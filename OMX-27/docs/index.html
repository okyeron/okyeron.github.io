OMX-27 MIDI controller and sequencer

OMX-27 is a DIY hardware MIDI controller and sequencer with RGB LED backlit mechanical key switches.

Features

  • Compact form factor
  • 27 RGB backlit mechanical key switches
  • 5 Potentiometers
  • 1 Encoder (w/ switch)
  • OLED display (32x128)
  • USB-MIDI
  • Hardware MIDI In (1/8" TRS stereo minijack, Type A/B auto-sensing) (v3)
  • Hardware MIDI Out (1/8" TRS stereo minijack, Type A/B switchable)
  • CV pitch and CV Gate outputs (1/8" TS mono minijack)
  • RP2040 microcontroller (v3)
  • USB-C for power/data (v3)
  • Optionally battery powered with lipo (v3)

Dimensions: 313mm x 65mm
