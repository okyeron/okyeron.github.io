Modes Overview

MI - MIDI keyboard

Sends USB-MIDI and hardware MIDI notes along with 5 CCâ€™s.
Sends CV pitch and CV gate (limited to 4 octave range).

CH - Chords

S1 - Sequencer 1 - single pattern

8 patterns of 16 steps
plays one pattern at a time
variable pattern length

S2 - Sequencer 2 - multi pattern

8 patterns of 16 steps
plays all patterns at the same time
variable pattern length (for each pattern)
(still lots to do in the sequencer code)

GR - Grids Sequencer

An adaptation of the Mutable Instruments "Topographic drum sequencer" module.

EL - Euclidian Sequencer

OM - Organelle Mother

The same as MI, but with modifications specifically for the Organelle Mother pd patch.
